# Laptop Price Prediction

## Overview
This project predicts laptop prices based on specifications such as RAM, processor, storage, GPU, and brand using Machine Learning techniques.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Features Used
- RAM
- SSD
- HDD
- Brand
- Processor
- GPU
- Screen Size

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Results
Achieved an R² score of 0.91 for laptop price prediction.

## Project Outcome
Built an interactive Streamlit application that allows users to estimate laptop prices based on configuration inputs.
